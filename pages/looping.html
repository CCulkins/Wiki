Looping

JavaScript loops offer a quick and easy way to do something repeatedly.

Code Example

          
          $(function(){
            var flavors = ["rocky-road", "pistacio", "cookiedough"];
            for (var index = 0; index < flavors.length; index += 1){
              $(".creamflav").append( "<li>" + flavors[index] + "</li>");
            };
